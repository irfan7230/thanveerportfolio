High-Performance Vedic Multiplier

This project focused on optimizing digital multiplication operations, critical for high-speed processors and DSP applications. By designing and implementing a Vedic multiplier in **Verilog/System Verilog**, we effectively replaced conventional hybrid full adders. This architectural change led to significant reductions in power consumption and boosted computational speed, showcasing an advanced understanding of low-power VLSI design techniques. The core challenge was balancing power efficiency with maximum throughput.

Technologies Used:

  • **Verilog/System Verilog:** For RTL design and behavioral modeling.
  • **EDA Tools:** Xilinx ISE/Vivado for synthesis and simulation.
  • **Key Concepts:** High-speed arithmetic, parallel processing, power optimization.

Neuromorphic Computing for Powerless Medical Sensors

This innovative project explored neuromorphic principles for ultra-low-power medical devices. I designed and implemented a circuit using **Verilog** that converts analog signals from medical sensors (like ECG, EEG) into digital spikes. This spiking data is then fed into a **Spiking Neural Network (SNN)** for real-time, adaptive monitoring. The primary goal was to create self-sustaining, powerless wearables, enabling continuous health monitoring without frequent battery changes.

Technologies Used:

  • **Verilog:** For comparator and spike encoder logic.
  • **Conceptual SNN Integration:** For pattern recognition.
  • **Key Concepts:** Analog-to-digital conversion, event-driven computing, biomedical signal processing.

*Image: Example of analog input converted to digital spikes.*

FSM Vending Machine Controller

This project involved the complete design and rigorous verification of a state machine for a coin-operated electronic newspaper vending machine. Using **Verilog HDL**, I implemented a Finite State Machine (FSM) capable of accepting nickels and dimes, accurately tracking the deposited amount, and dispensing a newspaper upon reaching the required 15 cents. A critical part of the project was developing a comprehensive **test bench** to verify every state transition and input condition, ensuring robust and error-free operation.

Technologies Used:

  • **Verilog HDL:** For FSM implementation and combinational/sequential logic.
  • **Test Bench Development:** For thorough functional verification.
  • **Key Concepts:** Digital logic design, state machines, hardware testing.

*Image: Testbench output showing successful coin input and dispense sequence.*

Wireless Notice Board (Embedded)

This project focused on modernizing information dissemination in public and organizational settings. I developed a **wireless notice board system** featuring an **LCD display** seamlessly connected to a **Bluetooth module**. This setup allows users to wirelessly project notices and updates from any Bluetooth-enabled device (smartphone, PC) directly to the display. This eliminates the need for manual updates, providing an efficient, real-time communication solution ideal for schools, offices, and public announcement areas.

Technologies Used:

  • **Microcontroller (e.g., Arduino/ESP32):** For controlling the LCD and Bluetooth module.
  • **LCD Display:** For visual output.
  • **Bluetooth Module:** For wireless communication.
  • **Key Concepts:** Embedded systems, wireless communication, real-time displays.
